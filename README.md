# Browser Compatibility Checker

A Vue 3 application that checks and displays browser compatibility for modern web features.

## Features Checked

- WebGL2
- Web Audio API
- Local Storage
- Promises
- CSS Transforms
- CSS Transitions
- CSS Animations
- CSS Grid
- CSS Variables
- Flexbox
- Service Workers

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

### Add Icon Image

Place your icon image at `/public/images/Icon material-error-outline.png` or update the image path in `CompatibilityChecker.vue`.

- Vue 3
